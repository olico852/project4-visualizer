This is a real-time analytics dashboard for Wdditv2

Inspired by the many data analytic tools and marketing trends, I built a custom dashboard for a Node/Express project made during my General Assembly Web Dev Immersive course.

The dashboard is a separate build from Wddit and can be adapted for other projects.

Tech stack: MongoDB/Mongoose, NodeJS/Express. Libraries: D3.JS, jQuery, Bootstrap, Socket.io


How to use:

Head over to Wdditv2 and start the user journey as a guest. For best results, please launch it in your browser's private browsing/incognito mode.
Load Wddit Visualizer on a separate tab, preferably lined next to your Wdditv2 window so that you can observe the real-time changes.
1) Search for a post using unique keywords... you should see your query reflected on the Visualizer's Guests & Conversions tab.
2) Register as a user and this will update the conversion pie chart in Guests & Conversions tab.
3) As a user, post new articles and comment on other's post. You will see your user activity reflected on User Activity tab.
